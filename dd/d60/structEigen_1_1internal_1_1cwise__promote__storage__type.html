Detailed Description

template<typename A, typename B, typename Functor>
struct Eigen::internal::cwise_promote_storage_type< A, B, Functor >

Specify the "storage kind" of applying a coefficient-wise binary operations between two expressions of kinds A and B respectively. The template parameter Functor permits to specialize the resulting storage kind wrt to the functor. The default rules are as follows:

A op A -> A
A op dense -> dense
dense op B -> dense
sparse op dense -> sparse
dense op sparse -> sparse
