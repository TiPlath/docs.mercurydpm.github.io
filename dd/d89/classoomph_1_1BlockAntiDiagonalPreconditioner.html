Detailed Description

template<typename MATRIX>
class oomph::BlockAntiDiagonalPreconditioner< MATRIX >

Block "anti-diagonal" preconditioner, i.e. same as block diagonal but along the other diagonal of the matrix (top-right to bottom-left).

const unsigned &  nblock 
) const
inlineprotectedvirtual

This is a helper function to allow us to implement BlockAntiDiagonal using BlockDiagonal preconditioner and only changing this function. Get the second index for block number i. Obviously for a diagonal preconditioner we want the blocks (i,i). For anti diagonal we will want blocks (i, nblock - i - 1).
