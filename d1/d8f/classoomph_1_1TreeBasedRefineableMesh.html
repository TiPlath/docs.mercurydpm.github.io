Detailed Description

template<class ELEMENT>
class oomph::TreeBasedRefineableMesh< ELEMENT >

///////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////// Templated base class for refineable meshes. The use of the template parameter is required only for creating new elements during mesh adaptation. This class overloaded the template-free inteface to the function split_elements_if_required() to make use of the template parameter. All refineable meshes should inherit directly from TreeBasedRefineableMesh<ELEMENT>
