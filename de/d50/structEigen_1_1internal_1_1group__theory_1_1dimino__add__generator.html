Detailed Description

template<template< typename, typename > class Multiply, template< typename, typename > class Equality, typename id, typename elements, typename generators_done, typename current_generator, bool redundant>
class Eigen::internal::group_theory::dimino_add_generator< Multiply, Equality, id, elements, generators_done, current_generator, redundant >

Enlarge the group by adding a new generator.

It accepts a boolean parameter that determines if the generator is redundant, i.e. was already seen in the group. In that case, it reduces to a no-op.
