MercuryDPM Reference Manual

This is the reference manual of MercuryDPM, a free, open source software package for performing discrete particle simulations. The aim of this manual is to provide all necessary information to guide a new user through the installation process, teach the user how to run, visualisa and analyse MercuryDPM simulations, and how to create new MercuryDPM codes to model a system of your choice.

The manual is split into three parts:

  • Installing MercuryDPM
  • Getting started
  • Fun with MercuryDPM (Tutorials)
  • Write Your Own Code
  • Coarse graining: From discrete particles to continuum fields
  • Visualising Your Results
  • Coupling with oomph-lib
  • Guidelines for Users and Developers
  • Contact Models in MercuryDPM
  • For Developers

For more information regarding the code, visit http://mercurydpm.org/
