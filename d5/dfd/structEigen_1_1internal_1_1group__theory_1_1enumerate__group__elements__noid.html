class Eigen::internal::group_theory::enumerate_group_elements_noid< Multiply, Equality, id, generators, initial_global_flags >

Helper template that implements group element enumeration.

This is a helper template that implements the actual enumeration of group elements. This has been split so that the list of generators can be cleansed of the identity element before performing the actual operation.
