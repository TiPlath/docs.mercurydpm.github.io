Detailed Description

Simulates a very thin rotating drum, to test liquid bridge migration. You can change some of the default parameters via the command line interface. E.g., ./RotatingDrumWet -bond 5 -distributeLiquidInhomogeneously sets surface tension such that bond number is 5 and changes to an initially inhomogeneous liquid distribution.
