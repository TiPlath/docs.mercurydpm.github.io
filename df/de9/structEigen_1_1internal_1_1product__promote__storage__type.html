template<typename A, typename B, int ProductTag>
struct Eigen::internal::product_promote_storage_type< A, B, ProductTag >

Specify the "storage kind" of multiplying an expression of kind A with kind B. The template parameter ProductTag permits to specialize the resulting storage kind wrt to some compile-time properties of the product: GemmProduct, GemvProduct, OuterProduct, InnerProduct. The default rules are as follows:

K * K -> K
dense * K -> dense
K * dense -> dense
diag * K -> K
K * diag -> K
Perm * K -> K
K * Perm -> K
