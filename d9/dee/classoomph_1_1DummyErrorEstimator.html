Detailed Description

///////////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////////// Dummy error estimator, allows manual specification of refinement pattern by forcing refinement in regions defined by elements in a reference mesh.
