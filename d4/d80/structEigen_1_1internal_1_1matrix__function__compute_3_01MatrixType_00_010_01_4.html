Detailed Description

template<typename MatrixType>
struct Eigen::internal::matrix_function_compute< MatrixType, 0 >

Partial specialization of MatrixFunction for real matrices.

This converts the real matrix to a complex matrix, compute the matrix function of that matrix, and then converts the result back to a real matrix.
