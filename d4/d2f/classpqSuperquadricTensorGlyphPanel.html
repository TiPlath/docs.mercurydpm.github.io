Detailed Description

Custom panel for SuperquadricTensorGlyph filter. This panel is only provided to add capabilities to hide/gray out GUI elements in case they do not apply and to link the checkbox FixedThetaPhiRoundness to GammaRoundness/ThetaRoundness/PhiRoundness.
