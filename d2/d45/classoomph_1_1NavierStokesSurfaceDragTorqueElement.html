Detailed Description

template<class ELEMENT>
class oomph::NavierStokesSurfaceDragTorqueElement< ELEMENT >

A class of elements that allow the determination of the drag and toque, relative to a given centre of rotation, along a domain boundary. The element operates as a FaceElement and attaches itself to a bulk element of the type specified by the template argument.
